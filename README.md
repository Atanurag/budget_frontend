## 游닇 Quick Reference

**Where Components Come From:**
- Green = `apps/web/src/components/` in scheduling app
- Examples: `ui/button/button.tsx`, `ui/input/input.tsx`, `ui/table/table.tsx`

**Smart/Dumb Pattern:**
- Every new component needs 2 files
- Example: `UserGreeting.tsx` (dumb) + `UserGreetingContainer.tsx` (smart)

**Libraries Needed:**
- `@dnd-kit/core` - for drag and drop
- `react-dropzone` - for file uploads
